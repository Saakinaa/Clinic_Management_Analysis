## Project Overview

This project focuses on the analysis of fictional healthcare data from a clinical environment. The dataset includes structured information on departments, staff, patients, and daily operational metrics.

The objective is to demonstrate how data analysis can support decision-making in healthcare systems by identifying trends, inefficiencies, and opportunities for improvement.

---

## Dataset Description

The dataset is organized into the following components:

* Departments: Information about hospital units (e.g., Emergency, Cardiology, Pediatrics)
* Staff: Details on healthcare personnel, roles, and department assignments
* Patients: Demographic and clinical information
* Daily Metrics: Operational indicators such as admissions, discharges, and workload

---

## Objectives

* Analyze patient distribution across departments
* Evaluate staff allocation and workload
* Identify trends in daily clinical activity
* Generate insights to improve operational efficiency

---

## Data Analysis Workflow

* Data cleaning and preprocessing
* Handling missing values and inconsistencies
* Exploratory Data Analysis (EDA)
* Data aggregation and grouping
* Interactive data visualization using Plotly
* Insight generation

---

## Key Analyses

* Patient flow across departments
* Staff workload distribution
* Daily admissions and discharge trends
* Department performance comparison

---

## Visualization Approach

All visualizations in this project are built using Plotly to provide:

* Interactive charts for better data exploration
* Dynamic filtering and zooming capabilities
* Clear and intuitive presentation of key metrics
